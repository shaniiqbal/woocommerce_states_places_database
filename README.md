<p>WordPress plugin that shows dropdowns for State and City Select for WooCommerce.</p>
<p>This WooCommerce plugin transforms the text input for states, the city or town. With this plugin you can provide a list of states and cities to be shown as a select dropdown.</p>
<p>This will be shown in checkout pages, edit addresses pages, shipping calculator, etc.</p>
